# 🧠 Sobre o Projeto

O **Dashboard Ninja** é uma plataforma de estudos gamificada criada para transformar o aprendizado de programação em uma experiência mais dinâmica, visual e motivadora.

O projeto foi desenvolvido com foco em:
- gamificação
- evolução de usuário
- experiência visual
- organização de estudos
- motivação através de progressão

A plataforma utiliza sistemas inspirados em RPG/anime para incentivar constância e evolução no aprendizado de programação.

# 🎯 Objetivo do Projeto

O principal objetivo do Dashboard Ninja é transformar os estudos em uma experiência:
- mais divertida
- mais organizada
- mais motivadora
- mais visual
- mais interativa

Além disso, o projeto também foi desenvolvido como prática avançada de:
- lógica de programação
- manipulação de DOM
- estruturação de sistemas
- UX/UI
- organização de projetos reais
